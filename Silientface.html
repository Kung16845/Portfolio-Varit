About Game Project

เกมจะมอบประสบการ์ณการเอาตัวรอดที่ตึงเครียด ที่ทุกปัญหาที่เจอจะมีทางออกหลายทางเสมอ เน้นมอบประสบการ์ณ Immersion ถึงสิ่งที่อยู่ในเกมให้ผู้เล่นดื่มด่ำไปกับบรรยากาศที่น่ากลัวไม่น่าไว้วางใจ และเมื่อผู้เล่นทำอะไรสำเร็จหรือเอาชนะอะไรบางอย่างได้ ผู้เล่นจะได้ประสบการ์ณให้สบายใจในช่วงเวลาหนึ่งเพราะศัตรูนั้นไม่ได้หายไปไหนมันยังอยู่มันแค่หนีไปซ่อนตัและมันะออกมาอีกครั้งแน่นอน การสำรวจฉากเพื่อให้ผู้เล่นมีทางเอาตัวรอดมากขึ้น

What I learned

    ผมได้พัฒนาทักษะการเขียนโค้ดที่เกี่ยวข้องกับการจัดการระบบต่างๆ ดังนี้

  • State Machine

    ผมได้ออกแบบและพัฒนาระบบ State Machine ที่ประกอบด้วย 5 สถานะ ได้แก่ State Hunting, State Listening, State Retreat, State Searching, และ State SearchingSound พร้อมกับการใช้ NavMeshPlus Package เพื่อช่วยสร้างเส้นทางที่สั้นที่สุดให้ศัตรู (Enemy) สามารถไล่ล่าผู้เล่นได้อย่างมีประสิทธิภาพตลอดทั้งเกม

  • Inventory Manager

    ผมได้ออกแบบและพัฒนาระบบ Inventory Manager ที่ช่วยให้ผู้เล่นสามารถเก็บ ย้าย และสลับไอเท็มได้อย่างง่ายดาย โดยระบบนี้แยกช่องเก็บของเฉพาะสำหรับอุปกรณ์ต่างๆ เช่น ช่องมือ และช่องสำหรับไฟฉาย เพื่อเพิ่มความสะดวกในการใช้งาน

  • Save Manager

    ผมได้พัฒนาระบบ Save Manager ในการเซฟและโหลดข้อมูลในเกม ด้วยการเก็บข้อมูลผ่านไฟส์ Json เพื่อให้ผู้เล่นสามารถกลับมาเล่นเกมต่อไปได้

    • โปรเจคนี้ช่วยให้ผมได้เรียนรู้การจัดการโค้ดที่ซับซ้อนและการเชื่อมโยงระบบต่างๆ เข้าด้วยกัน ซึงสิ่งเหล่านี้ทำให้ผมได้รู้ถึงข้อผิดพลาดในการที่ไม่ได้วางแผนให้ดี สิ่งที่ผมได้เรียนรู้คือความสำคัญของการวางแผนอย่างรอบคอบในการเขียนโค้ด การขาดการวางแผนที่ดีอาจทำให้โค้ดมีความยุ่งเหยิงและยากต่อการเข้าใจ การทำโปรเจคนี้จึงช่วยให้ผมตระหนักถึงการพัฒนาแนวคิด Clean Code เพื่อนำไปใช้ในโปรเจคต่อไป
