About Game Project

The game will provide a tense survival experience where every problem encountered will always have multiple solutions. It emphasizes providing an immersive experience, allowing players to be absorbed in the game's fearful and untrustworthy atmosphere. When players succeed in something or overcome an obstacle, they will experience a moment of relief. However, this relief is temporary because the enemies haven't disappeared; they've just retreated to hide and will certainly return. Exploring the scenes will give players more ways to survive.

What I learned

    I have developed coding skills related to managing various systems as follows:

  • State Machine

    I have designed and developed a State Machine system along with the NavMeshPlus Package to help create the shortest path for enemies to efficiently chase players throughout the game. This system controls AI actions, consisting of 5 states:

    State Hunting : AI will chase the player when detected.
    State Listening : AI will listen for the player's footsteps and move towards the AI nest closest to the player.
    State Retreat : AI will retreat when its health drops to a specified level.
    State Searching : AI will search for the player at various points in nearby rooms.
    State SearchingSound : AI will move towards the point where a sound is generated by the player's actions.

    This system enhances the challenge and realism of AI in the game by employing multiple strategies to track and pursue the player.

  • Inventory Manager

    I have designed and developed an Inventory Manager system that allows players to efficiently manage their items. Players can easily store, move, and swap items. This system includes separate storage slots for various equipment, including:ก่

    Hand slot : For items that players can use immediately in the game
    Flashlight slot : For flashlights that provide illumination during gameplay
    Bag slot : For storing items that players collect

    This system is designed to enhance convenience and streamline the use of various equipment within the game.

  • Save Manager

    I have developed a Save Manager system for saving and loading game data using JSON files, allowing players to resume their game progress.
