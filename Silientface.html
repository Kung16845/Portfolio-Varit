About Game Project

เกมจะมอบประสบการ์ณการเอาตัวรอดที่ตึงเครียด ที่ทุกปัญหาที่เจอจะมีทางออกหลายทางเสมอ เน้นมอบประสบการ์ณ Immersion ถึงสิ่งที่อยู่ในเกมให้ผู้เล่นดื่มด่ำไปกับบรรยากาศที่น่ากลัวไม่น่าไว้วางใจ และเมื่อผู้เล่นทำอะไรสำเร็จหรือเอาชนะอะไรบางอย่างได้ ผู้เล่นจะได้ประสบการ์ณให้สบายใจในช่วงเวลาหนึ่งเพราะศัตรูนั้นไม่ได้หายไปไหนมันยังอยู่มันแค่หนีไปซ่อนตัและมันะออกมาอีกครั้งแน่นอน การสำรวจฉากเพื่อให้ผู้เล่นมีทางเอาตัวรอดมากขึ้น

What I learned

    ผมได้พัฒนาทักษะการเขียนโค้ดที่เกี่ยวข้องกับการจัดการระบบต่างๆ ดังนี้

  • State Machine

    ผมได้ออกแบบและพัฒนาระบบ State Machine พร้อมกับการใช้ NavMeshPlus Package เพื่อช่วยสร้างเส้นทางที่สั้นที่สุดให้ศัตรู (Enemy) สามารถไล่ล่าผู้เล่นได้อย่างมีประสิทธิภาพตลอดทั้งเกม ในการควบคุมการกระทำของ AI ที่ประกอบด้วย 5 สถานะ ได้แก่

    State Hunting AI จะไล่ล่าผู้เล่นเมื่อพบเจอ
    State Listening AI จะคอยฟังเสียงฝีเท้าของผู้เล่นและเคลื่อนที่ไปยังรังของ AI ที่ใกล้กับผู้ล่น
    State Retreat AI จะล่าถอยเมื่อพลังชีวิตลดลงจนถึงจุดที่กำหนด
    State Searching AI จะค้นหาผู้เล่นตามจุดต่างๆ ในห้องที่อยู่ใกล้
    State SearchingSound AI จะเคลื่อนที่ไปยังจุดที่เกิดเสียงดังจากการกระทำของผู้เล่น

    ระบบนี้ช่วยเพิ่มความท้าทายและความสมจริงให้กับ AI ในเกม โดยใช้กลยุทธ์หลายรูปแบบในการติดตามและไล่ล่าผู้เล่น

  • Inventory Manager

    ผมได้ออกแบบและพัฒนาระบบ Inventory Manager ที่ช่วยให้ผู้เล่นสามารถจัดการไอเท็มได้อย่างมีประสิทธิภาพ ผู้เล่นสามารถเก็บ ย้าย และสลับไอเท็มได้อย่างง่ายดาย โดยระบบนี้มีการแยกช่องเก็บของเฉพาะสำหรับอุปกรณ์ต่างๆ ได้แก่

    ช่องมือ สำหรับใส่ไอเท็มที่ผู้เล่นสามารถใช้งานได้ทันทีในเกม
    ช่องไฟฉาย สำหรับใส่ไฟฉายที่ให้แสงสว่างระหว่างการเล่นเกม
    ช่องกระเป๋า สำหลับการใส่ Item ที่ผู้เล่นเก็บได้

    ระบบนี้ถูกออกแบบมาเพื่อเพิ่มความสะดวกและทำให้การใช้งานอุปกรณ์ต่างๆ ภายในเกมเป็นไปอย่างราบรื่น

  • Save Manager

    ผมได้พัฒนาระบบ Save Manager ในการเซฟและโหลดข้อมูลในเกม ด้วยการเก็บข้อมูลผ่านไฟส์ Json เพื่อให้ผู้เล่นสามารถกลับมาเล่นเกมต่อไปได้

    • โปรเจคนี้ช่วยให้ผมได้เรียนรู้การจัดการโค้ดที่ซับซ้อนและการเชื่อมโยงระบบต่างๆ เช่น State Machine, Inventory Manager, และ Save Manager เข้าด้วยกัน ซึงสิ่งเหล่านี้ทำให้ผมได้รู้ถึงข้อผิดพลาดในการที่ไม่ได้วางแผนให้ดี สิ่งที่ผมได้เรียนรู้คือความสำคัญของการวางแผนอย่างรอบคอบในการเขียนโค้ด การขาดการวางแผนที่ดีอาจทำให้โค้ดมีความยุ่งเหยิงและยากต่อการเข้าใจ การทำโปรเจคนี้จึงช่วยให้ผมตระหนักถึงการพัฒนาแนวคิด Clean Code เพื่อนำไปใช้ในโปรเจคต่อไป
