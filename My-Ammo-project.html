About Game Project

ผู้เล่นต้องให้ร่วมมือกันอย่างมีประสิทธิภาพ เพื่อวางกลยุทธ์และตัดสินใจท่ามกลางสถานการณ์ที่วุ่นวาย ผู้เล่นต้องบริหารจัดการการกระทำของตนอย่างรอบคอบ เพื่อชะลอการบุกของซอมบี้ และเก็บทรัพยากรจากคลังในระหว่างการโจมตี

What I learned

    ผมได้พัฒนาทักษะการเขียนโค้ดที่เกี่ยวข้องกับการจัดการระบบต่างๆ ดังนี้

  • CO-OP Manager

    ผมได้ออกแบบและพัฒนาระบบ Co-op ที่ช่วยให้ผู้เล่นสามารถเข้าร่วมเล่นด้วยกันผ่าน Lobby โดยในหน้าต่าง Lobby จะแสดงรายชื่อผู้เล่นทั้งหมด และมีฟีเจอร์ที่เฉพาะผู้ที่เป็น Host เท่านั้นที่จะสามารถกดปุ่มเริ่มเกมได้ นอกจากนี้ ยังมีปุ่ม Ready เพื่อให้ผู้เล่นทุกคนกดพร้อมก่อนเริ่มเกม ระบบนี้ถูกพัฒนาด้วย NetCode

  • Spawner Manager

    ผมได้ออกแบบและพัฒนาระบบ Enemy Spawner และ Item Spawner สำหรับการเล่น Co-op โดยระบบ Enemy Spawner จะทำหน้าที่สุ่มตำแหน่งจุดเกิดของศัตรูตามช่วงเวลาที่กำหนด เพิ่มความท้าทายให้กับผู้เล่น ในขณะที่ระบบ Item Spawner จะสุ่มวางไอเท็มในตำแหน่งต่างๆ บนแผนที่ เพื่อให้ผู้เล่นเก็บและนำไปใช้ในการต่อสู้ ทำให้การเล่นเกมมีความหลากหลายและน่าตื่นเต้นยิ่งขึ้น

    • โปรเจคนี้ทำให้ผมได้เรียนรู้และเข้าใจการพัฒนาเกมแบบ Co-op โดยเฉพาะการใช้ NetCode ในการจัดการระบบผู้เล่นหลายคน ผมได้เรียนรู้ถึงความสำคัญของการคำนึงถึงปัจจัยต่างๆ ที่ส่งผลต่อประสบการณ์ผู้เล่น เช่น latency ที่ส่งผลต่อความลื่นไหลของเกม การประมวลผลข้อมูลแบบเรียลไทม์ และการอัปเดตแผนที่เมื่อมีการเปลี่ยนแปลงในฉาก สิ่งเหล่านี้ทำให้ผมตระหนักถึงความซับซ้อนของปัญหาในการพัฒนาเกมที่ต้องรองรับผู้เล่นหลายคน และช่วยให้ผมสามารถวางแผนการเขียนโค้ดได้อย่างมีประสิทธิภาพยิ่งขึ้นในอนาคต
