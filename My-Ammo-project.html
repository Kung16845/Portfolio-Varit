About Game Project

ผู้เล่นต้องให้ร่วมมือกันอย่างมีประสิทธิภาพ เพื่อวางกลยุทธ์และตัดสินใจท่ามกลางสถานการณ์ที่วุ่นวาย ผู้เล่นต้องบริหารจัดการการกระทำของตนอย่างรอบคอบ เพื่อชะลอการบุกของซอมบี้ และเก็บทรัพยากรจากคลังในระหว่างการโจมตี

What I learned

    ผมได้พัฒนาทักษะการเขียนโค้ดที่เกี่ยวข้องกับการจัดการระบบต่างๆ ดังนี้

  • CO-OP Manager

    ผมได้ออกแบบและพัฒนาระบบ Co-op ที่ผู้เล่นสามารถเข้าร่วมเล่นด้วยกันผ่าน Lobby ในหน้าต่าง Lobby จะแสดงรายชื่อผู้เล่นทั้งหมด และเฉพาะผู้ที่เป็น Host เท่านั้นที่จะสามารถกดปุ่มเริ่มเกมได้ นอกจากนี้ ยังมีปุ่ม Ready เพื่อให้ผู้เล่นทุกคนกดพร้อมก่อนเริ่มเกม ด้วย NetCode

  • Spawner Manager

    ผมได้ออกแบบและพัฒนาระบบ Enemy Spawner และ Item Spawner สำหรับการเล่น CO-OP โดยระบบ Enemy Spawner จะทำหน้าที่สุ่มต่ำแหน่งจุดเกิดของศัตรูตามระยะเวลาที่กำหนด ในขณะที่ระบบ Item Spawner จะสุ่มวางไอเท็มตามตำแหน่งต่างๆ เพื่อให้ผู้เล่นเก็บใช้ในการต่อสู้

    • โปรเจคนี้ช่วยให้ผมได้เข้าใจ NetCode มากขึ้นสำหรับการสร้างระบบ CO-OP เพื่อให้ผู้เล่นสามารถเล่นร่วมมือกันได้อย่างมีประสิทธิ์ภาพ ผมได้เรียนรู้ว่าการสร้างเกมที่รองรับผู้เล่นหลายคนต้องคำนึงถึงปัจจัยสำคัญหลายอย่าง เช่น ความหน่วงในการเล่นเกม (latency) การประมวลผลแบบเรียลไทม์ และการอัปเดตแผนที่ (Map) เมื่อมีการเปลี่ยนแปลงในฉาก (Scene) และสิ่งเหล่านี้ทำให้ผมรู้ข้อผิดพลาดในสิ่งที่ผมได้เขียนลงไป
