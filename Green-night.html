About Game Project

It is a survival and base defense game where players must manage resources wisely, create equipment, and employ planning and strategies such as scavenging, crafting, and trading to ensure their survival.

What I learned

    I have developed coding skills related to managing various systems as follows:

  • Building Manager

    I have designed and developed a system that can create structures in three sizes: small, medium, and large. Each size requires different resources and construction time, which means players need to plan their resource usage and time management efficiently.

  • Time Manager

    I have developed a day and time counting system in the game, which works in conjunction with the construction system. Different buildings will take varying amounts of time to construct. dditionally, the system is connected to resource management that can be used for in-game trading.

    • This project has helped me practice complex system coding skills and efficiently integrate multiple systems to work together. Although the coding in this instance may not be perfect in terms of Clean Code, I am committed to continuously improving my coding skills to make the code more organized, easy to understand, and suitable for team collaboration.
