About Game Project

เป็นเกมเอาตัวรอดและป้องกันฐานที่ผู้เล่นต้องบริหารทรัพยากรในการสร้างฐานทัพและเตรียมตัวในการรับมือกับซ้อมบี้ที่จะบุก

Project Info

Role : Game Developer

Team size : 3 คน

Time Frame : WIP

Engine : Unity


What I learned

ผมได้พัฒนาทักษะการเขียนโค้ดที่เกี่ยวข้องกับการจัดการระบบต่างๆ ดังนี้

  • Building Manager – ผมได้ออกแบบและพัฒนาระบบที่สามารถสร้างสิ่งปลูกสร้างได้ในสามขนาด ได้แก่ เล็ก, กลาง และใหญ่ โดยแต่ละขนาดมีการใช้ทรัพยากรและเวลาในการก่อสร้างที่แตกต่างกัน
  • Time Manager – ผมได้สร้างระบบนับวันและเวลา และใช้งานร่วมกันกับการสร้างสิ่งปลูกสร้างที่ต้องใช้เวลา มีระบบทรัพยากรในการใช้ซื้อขาย.

โปรเจคนี้ช่วยให้ผมได้ฝึกฝนทักษะการเขียนโค้ดเชิงระบบที่ซับซ้อนและสามารถเชื่อมโยงหลายๆ ระบบให้ทำงานร่วมกันได้อย่างมีประสิทธิภาพ แม้ว่าการเขียนโค้ดในครั้งนี้อาจจะยังไม่สมบูรณ์แบบในแง่ของ Clean Code แต่ผมมุ่งมั่นที่จะพัฒนาทักษะการเขียนโค้ดให้ดีขึ้นอย่างต่อเนื่อง เพื่อให้โค้ดมีความเป็นระเบียบ เข้าใจง่าย และเหมาะสมสำหรับการทำงานร่วมกันในทีม
